Tutorial Singkat Penggunaan wxMaxima

1 Pendahuluan

Pada tutorial singkat ini akan dipelajari penggunaan dasar wxMaxima. Maxima merupakan Sistem Aljabar Komputer (Computer Algebra System, CAS), serupa dengan CAS lain seperti Mathematica, Maple dan lainnya. Adapun wxMaxima adalah antar-muka grafis (GUI, graphical user interface) untuk mengoperasikan Maxima sehingga lebih mudah dan praktis digunakan.

2 Penggunaan Dasar

Berikut ini adalah contoh kalkulasi sederhana, dalam hal ini Maxima berperan seperti kalkulator saintifik biasa.
Di bawah ini adalah sel masukan (input cell) dengan operasi penjumlahan sederhana dua bilangan. Letakkan kursor pada sel tersebut kemudian tekan SHIFT-ENTER (tekan tombol SHIFT dan tombol ENTER secara bersamaan) untuk memproses kalkulasi tersebut

Hasil kalkulasi akan ditampilkan pada sel keluaran (output cell).
Akan terlihat bahwa sel masukan (input cell) ditandai dengan penanda (%i1), sedangkan sel keluaran (output cell) ditandai dengan penanda (%o1). Angka 1 pada penanda sel masukan dan sel keluaran merupakan nomor urut sel.

Berikut diberikan lagi beberapa kalkulasi. Untuk memproses kalkulasi dan mendapatkan hasilnya, lakukan hal yang sama dengan yang sebelumnya yaitu letakkan kursor pada sel masukan dan kemudian tekan tombol SHIFT-ENTER bersamaan.

Pada sel masukan yang ditunjukkan di atas, terdapat 5 baris perintah (proses kalkulasi) yang kita perintahkan kepada Maxima untuk dihitung. Setiap baris perintah haruslah diakhiri dengan tanda titik koma ";" atau tanda dollar "$". Jika baris perintah diakhiri dengan tanda ";", maka Maxima akan menampilkan hasil kalkulasi baris tersebut pada sel keluaran. Sedangkan jika suatu baris diakhiri dengan tanda "$", maka Maxima tidak menampilkan hasil kalkulasi baris tersebut pada sel keluaran. Perhatikan bahwa satu sel masukan yang terdiri dari 5 baris perintah tersebut (yang ditandai dengan penanda (%i2)) akan menghasilkan 5 sel keluaran (yang ditandai dengan penanda (%o2) sampai (%o6)). Perhatikan bahwa proses kalkulasi baris 4 (yaitu "1/3") dan baris 5 (yaitu "1.0/5.0") memberikan hasil yang berbeda. Hal ini dikarenakan Maxima tetap berusaha menampilkan bilangan dalam bentuk yang sederhana (tidak menampilkan hasil numerik). Jadi bilangan rasional 1/3 akan tetap ditampilkan 1/3, bilangan sqrt(2) akan tetap ditampilkan sqrt(2). Sedangkan bentuk "1.0/3.0" pada baris 5 menunjukkan bahwa hasil yang diinginkan adalah bilangan bertipe floating point. Tipe floating point ini dapat digunakan untuk menampilkan hasil numerik jika diinginkan. Misalnya contoh berikut
(%i8) sqrt ( 2 * %pi ) ;
float ( % ) ;

\[\operatorname{ }\sqrt{2} \sqrt{\ensuremath{\pi} }\]

\[\operatorname{ }2.506628274631001\]

Pada baris kedua yang berupa perintah "float(%);" digunakan simbol "%". Simbol ini dapat digunakan untuk mengacu pada hasil proses sebelumnya.
Bilangan yang dihasilkan pada proses di atas dapat dirujuk pada proses perhitungan selanjutnya dengan menggunakan penanda (%o7) ataupun (%o8)

Dalam proses perhitung, bilangan dan ekspresi dapat disimpan dalam bentuk variabel. Dalam pendefinisian variabel, format yang digunakan adalah sebagai berikut
"nama_variabel:nilai;"
yang berarti menyimpan "nilai" dengan nama "nama_variabel".

Perhatikan contoh berikut
(%i12) jejari : 10 $
tinggi : 100 $
luas : %pi * jejari ^ 2 ;
volume : luas * tinggi ;

\[\operatorname{(luas) }100 \ensuremath{\pi} \]

\[\operatorname{(volume) }10000 \ensuremath{\pi} \]

Untuk menampilkan hasil numeriknya, sebagai berikut
(%i14) float ( luas ) ; float ( volume ) ;

3 Penggunaan Lebih Lanjut

Pada bagian terdahulu telah diperkenalkan penggunaan dasar Maxima untuk kalkulasi (perhitungan) sederhana seperti penjumlahan, perkalian, pembagian. Yang berarti Maxima digunakan sebagai kalkulator saintifik biasa. Maxima mempunyai kemampuan lebih jauh yang utamanya untuk proses perhitungan simbolik. Misalnya saja kita dapat menggunakan Maxima untuk menghitung integral simbolik (baik integral tertentu ataupun integral tak tentu) seperti berikut
(%i16) integrate ( sin ( x ) , x ) ;
integrate ( sin ( x ) , x , 0 , %pi ) ;

\[\operatorname{ }-\cos{(x)}\]

\[\operatorname{ }2\]

Contoh berikutnya adalah pengintegral fungsi yang didefinisikan terlebih dulu
(%i20) f ( x ) : = x ^ 2 + a $
f ( 5 ) ;
f ( 5 ) , a = - 5 ;
integrate ( f ( var ) , var ) ;

\[\operatorname{ }a+25\]

\[\operatorname{ }20\]

\[\operatorname{ }\frac{{{\ensuremath{\mathrm{var}}}^{3}}}{3}+a\, \ensuremath{\mathrm{var}}\]

Kadangkala Maxima memerlukan informasi tambahan untuk menyelesaikan suatu perhitungan dan meminta kita untuk menjawab pertanyaan. Pertanyaan "Is a positive or negative?"
Jawabannya dapat ditulis lengkap, atau bisa juga hanya huruf pertama ("p" atau "n").
Misalnya seperti berikut
(%i21) integrate ( 1 / ( x ^ 2 + a ) , x ) ;

\[\]\[\mbox{}\\"Is "a" positive or negative?"p;\]

\[\operatorname{ }\frac{\operatorname{atan}\left( \frac{x}{\sqrt{a}}\right) }{\sqrt{a}}\]

Kita dapat juga memberi tahu terlebih dahulu kepada Maxima nilai suatu besaran (sebelum ditanyakan). Gunakan fungsi "assume" untuk hal ini. Dan untuk menghapus asumsi digunakan fungsi "forget", sebagaimana contoh berikut ini.

(Sebagai catatan penting: untuk mendapatkan informasi lengkap tentang suatu fungsi dalam Maxima, kita dapat mengakses file bantuan dengan menekan tombol F1)
(%i24) assume ( a > 0 ) $
integrate ( 1 / ( x ^ 2 + a ) , x ) ;
forget ( a > 0 ) $

\[\operatorname{ }\frac{\operatorname{atan}\left( \frac{x}{\sqrt{a}}\right) }{\sqrt{a}}\]

4.2 Aljabar Linier, matriks

Untuk menampilkan matriks, digunakan perintah "matrix". Sebuah matriks dapat berisi ekspresi non bilangan (misalnya fungsi atau variabel). Perkalian matriks (inner product) menggunakan perintah "." Untuk memperoleh invers matriks, digunakan perintah "invert".
(%i30) A : matrix ( [ 1 , - 1 ] ,
         [ 1 , sin ( c ) ] ) ;
B : invert ( A ) ;

A . B ;
ratsimp ( A . B ) ;

\[\operatorname{(A) }\begin{pmatrix}1 & -1\\ 1 & \sin{(c)}\end{pmatrix}\]

\[\operatorname{(B) }\begin{pmatrix}\frac{\sin{(c)}}{\sin{(c)}+1} & \frac{1}{\sin{(c)}+1}\\ -\frac{1}{\sin{(c)}+1} & \frac{1}{\sin{(c)}+1}\end{pmatrix}\]

\[\operatorname{ }\begin{pmatrix}\frac{\sin{(c)}}{\sin{(c)}+1}+\frac{1}{\sin{(c)}+1} & 0\\ 0 & \frac{\sin{(c)}}{\sin{(c)}+1}+\frac{1}{\sin{(c)}+1}\end{pmatrix}\]

\[\operatorname{ }\begin{pmatrix}1 & 0\\ 0 & 1\end{pmatrix}\]

Pada baris terakhir digunakan perintah "ratsimp" untuk menyederhanakan hasil perkalian "A.B". Maxima memiliki beberapa fungsi berbeda untuk menyederhanakan ekspresi, bergantung keperluan. Selain "ratsimp" ada juga "radcan", "trigsimp", "trigreduce", dlsb.

Turunan suatu fungsi diperoleh dengan perintah "diff"
(%i39) h1 ( x ) : = sin ( x ) * %e ^ ( 2 * x ) ;

\[\operatorname{ }\operatorname{h1}(x)\operatorname{:=}\sin{(x)} {{\% e}^{2 x}}\]

\[\operatorname{(hx1) }2 {{\% e}^{2 x}} \sin{(x)}+{{\% e}^{2 x}} \cos{(x)}\]

\[\operatorname{(hxx1) }3 {{\% e}^{2 x}} \sin{(x)}+4 {{\% e}^{2 x}} \cos{(x)}\]

Apapun yang kita ketikkan di antara tanda /* dan */ tidak akan dievaluasi oleh Maxima, oleh karenanya ini dapat digunakan untuk menuliskan keterangan atau komentar atau catatan. Hal ini ditunjukkan seperti pada baris kedua dan ketiga di atas.
Turunan parsial untuk suatu fungsi multivariabel juga dapat diperoleh, contohnya sebagai berikut
(%i44) z ( x , y ) : = x ^ 2 * y + %e ^ ( 2 * y ) ;
zx : ' diff ( z ( x , y ) , x , 1 ) $ /*turunan pertama z terhadap variabel x*/
zy : ' diff ( z ( x , y ) , y , 1 ) $ /*turunan pertama z terhadap variabel y*/
zx = diff ( z ( x , y ) , x , 1 ) ;
zy = diff ( z ( x , y ) , y , 1 ) ;

\[\operatorname{ }\operatorname{z}\left( x\operatorname{,}y\right) \operatorname{:=}{{x}^{2}} y+{{\% e}^{2 y}}\]

\[\operatorname{ }\frac{d}{d x} \left( {{\% e}^{2 y}}+{{x}^{2}} y\right) =2 x y\]

\[\operatorname{ }\frac{d}{d y} \left( {{\% e}^{2 y}}+{{x}^{2}} y\right) =2 {{\% e}^{2 y}}+{{x}^{2}}\]

4.4 Persamaan Differensial

Untuk memperoleh solusi persamaan differensial, Maxima mempunyai fungsi "ode2"
Misalnya untuk persamaan differensial orde 2 yang dinyatakan dengan
y''(t) + omega^2 * y(t) = 0
dengan omega > 0
(%i54) pdb : ' diff ( y , t , 2 ) + omega ^ 2 * y = 0 ;
solum : ode2 ( pdb , y , t ) ; /*solusi umum*/

\[\operatorname{(pdb) }\frac{{{d}^{2}}}{d {{t}^{2}}} y+{{omega}^{2}} y=0 \]\[\mbox{}\\"Is "omega" zero or nonzero?"n;\]

\[\operatorname{(solum) }y=\ensuremath{\mathrm{\% k1}} \sin{\left( omega t\right) }+\ensuremath{\mathrm{\% k2}} \cos{\left( omega t\right) }\]

Untuk kasus initial value problem, solusi khusus dapat diperoleh menggunakan "ic2".
Misalnya syarat yang diberikan adalah y(t=0)=amp dan y'(t=0)=0
(%i55) ic2 ( solum , t = 0 , y = amp , ' diff ( y , t ) = 0 ) ;

\[\operatorname{ }y=\ensuremath{\mathrm{amp}} \cos{\left( omega t\right) }\]
