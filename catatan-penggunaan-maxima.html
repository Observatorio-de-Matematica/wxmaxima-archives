3 Fungsi, Variabel dan Persamaan

3.1 Variabel

Huruf yang diketikkan secara otomatis akan dikenali sebagai variabel. Misalnya x, s, p, dlsb. Jika ingin mendefinsikan suatu simbol dengan nilai tertentu (konstanta), misalnya ingin digunakan nilai R=3.5 dalam suatu perhitungan, maka diset agar nilai R sama dengan 3.5, dengan menggunakan perintah ":"
(%i47) R : 3 . 5 ;

\[\operatorname{(R) }3.5\]

(%i48) %pi * R ^ 2 ;

\[\operatorname{ }12.25 \ensuremath{\pi} \]

(%i49) A : %pi * R ^ 2 ;

\[\operatorname{(A) }12.25 \ensuremath{\pi} \]

(%i50) A ;

\[\operatorname{ }12.25 \ensuremath{\pi} \]

(%i51) float ( A ) ;

\[\operatorname{ }38.4845\]

(%i52) R_2 = 3 . 5 ;

\[\operatorname{ }{R_2}=3.5\]

(%i53) A_2 : %pi * R_2 ^ 2 ;

\[\operatorname{(A\_ 2) }\ensuremath{\pi} {{{R_2}}^{2}}\]

(%i54) float ( A_2 ) ;

\[\operatorname{ }3.14159 {{{R_2}}^{2}}\]

Variabel yang telah didefinisikan dapat dihapus dengan menggunakan perintah "kill"
(%i55) kill ( A ) ;

\[\operatorname{ }\ensuremath{\mathrm{done}}\]

(%i56) kill ( A_2 ) ;

\[\operatorname{ }\ensuremath{\mathrm{done}}\]

3.3 Persamaan

Mendefinisikan persamaan dapat dilakukan menggunakan tanda sama dengan ("=") sebagai berikut
(%i11) h ( x ) : = tan ( x ) ; h ( x ) = 0 ;

\[\operatorname{ }\operatorname{h}(x)\operatorname{:=}\tan{(x)}\]

\[\operatorname{ }\tan{(x)}=0\]

Atau dapat juga dengan langsung memberi label pada persamaan tersebut
(%i12) E : x ^ 2 = 4 ;

\[\operatorname{(E) }{{x}^{2}}=4\]

Panggil persamaan yang telah diberi label dengan "E" tersebut
(%i13) E ;

\[\operatorname{ }{{x}^{2}}=4\]

(%i14) F : 2 * x - 7 = 5 ;

\[\operatorname{(F) }2 x-7=5\]

Dua persamaan tersebut dapat dioperasikan secara aritmatika (misalnya dijumlah)
(%i15) E + F ;

\[\operatorname{ }{{x}^{2}}+2 x-7=9\]

atau operasi lainnya, misalnya perkalian, yang bisa saja diberi label baru
(%i16) G : 2 * E ;

\[\operatorname{(G) }2 {{x}^{2}}=8\]

(%i17) H : E * F ;

\[\operatorname{(H) }{{x}^{2}} \left( 2 x-7\right) =20\]
